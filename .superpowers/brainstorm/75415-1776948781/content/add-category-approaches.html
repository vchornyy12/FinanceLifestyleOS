A — Toggle + inline subcategories

A "Main category" checkbox unlocks an inline subcategory list inside the same form. Add names one by one. All saved in one submit.

Pros

  • One form, one submit
  • Clear visual grouping

Cons

  • Form gets taller
Step 1 of 2 — Category details

B — Two-step wizard

Step 1: create the parent (name/type/color). Step 2: add subcategories. Can skip step 2 to create a standalone category.

Pros

  • Each step focused
  • Skip keeps it simple

Cons

  • Two round-trips
  • More complex flow

C — Always-visible subcategory chips

Subcategories section always shown at the bottom of the form. No toggle needed — just type and press enter to add chips. Having subcategories implicitly makes it a main category.

Pros

  • Fewest clicks
  • No mode-switching

Cons

  • Form always tall
  • Chips UX is new pattern
